# AI Document Translator

A premium PDF translation application using React and Groq API.

## Features
- **PDF Translation**: Upload any PDF and translate it to multiple languages while maintaining basic structure.
- **Premium UI**: Glassmorphism design with smooth animations.
- **Powered by Groq**: High-quality AI translations using Llama 3.3.

## Setup

### Backend
1. Go to `backend` folder.
2. Open `.env` and add your `GROQ_API_KEY`.
3. Run `npm install` (if not already done).
4. Run `npm run dev`.

### Frontend
1. Go to `frontend` folder.
2. Run `npm install` (if not already done).
3. Run `npm run dev`.

## Technologies
- **Frontend**: Vite, React, TypeScript, Framer Motion, Lucide React, Axios.
- **Backend**: Node.js, Express, Multer, Groq SDK, PDF-Lib, PDF-Parse.
